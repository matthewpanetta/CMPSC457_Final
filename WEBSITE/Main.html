Description

This project is a civilization simulation where the user can gather resources and make various buildings. The plane that the buildings are made on is comprised of "tiles" that contain three randomly generated amounts of resources. There is wood, stone, and trees. The more of a resource the "tile" has, the better the output of the building that is placed on that "tile". The buildings then output a resource depending on their type.
